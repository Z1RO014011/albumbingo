# Album Bingo

一个围绕专辑封面联想玩法做的 5x5 Bingo 小游戏。

玩家先随机拿到一张由视觉词、情绪词、文化感受词混合组成的 Bingo 卡，再为每个格子搜索并放入自己认为最贴切的专辑封面。连成横线、竖线或对角线后就会触发 `BINGO!`。

## 功能简介

- 随机生成 5x5 Album Bingo 提示词卡片
- 提示词混合视觉、氛围、文化语境三类线索
- 点击格子后可搜索专辑并直接放入当前格子
- 已放入的专辑支持替换和清除
- 自动检测横线、竖线、对角线是否达成 Bingo
- 搜索失败时会给出状态提示
- 为没有封面的结果生成兜底封面图
- 适配桌面端与移动端浏览

## 本地运行

项目当前是纯前端页面，主入口文件是 `index.html`。

推荐在项目目录启动一个本地静态服务器：

```bash
python3 -m http.server 8080
```

然后在浏览器打开：

```text
http://127.0.0.1:8080
```

## 使用方式

1. 打开页面后，先查看当前 5x5 提示词卡片
2. 点击任意一个格子，打开专辑搜索弹层
3. 输入专辑名、歌手名或关键词进行搜索
4. 点击结果卡片，把对应专辑放入当前格子
5. 重复放置，直到连成一条横线、竖线或对角线
6. 如果想重玩，可以使用“重新开始”或“随机生成卡片”

## 搜索说明

- 当前默认使用 iTunes 搜索专辑数据
- 搜索结果会优先尝试匹配更准确的专辑与歌手信息
- 如果某些网络环境下搜索失败，页面会显示失败提示，但棋盘与交互仍可继续使用
